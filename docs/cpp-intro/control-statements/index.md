# Control Statements

Control statements direct the flow of a program.

## Examples

    int x = 10;
    if (x > 0) {
        println("Positive");  // Conditional execution
    }

    for (int i = 0; i < 5; i++) {
        println(i);  // Loops 0 to 4
    }
