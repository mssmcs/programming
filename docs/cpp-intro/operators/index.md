# Operators

Operators perform operations on values and variables.

## Types
- Arithmetic: `+`, `-`, `*`, `/`, `%`
- Relational: `==`, `!=`, `<`, `>`, `<=`, `>=`
- Logical: `&&`, `||`, `!`

Example:

    int a = 10, b = 5;
    int sum = a + b;      // 15
    bool isEqual = a == b;  // false
