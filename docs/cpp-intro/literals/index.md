# Literals

Literals are fixed values written directly in code.

## Examples
- Integer: `42`, `0xFF` (hexadecimal)
- Floating-point: `3.14`, `2.5e-3` (scientific notation)
- Character: `'a'`, `'
'` (newline)
- String: `"Hello, world!"`

Usage:

    int x = 100;      // Integer literal
    double y = 9.8;   // Floating-point literal
